Roadmap
==================================================

## 1.0 Sudoku Game

This version is based in providing the basic logic for playing the Sudoku games

- Sudoku puzzles 6x6 and 9x9
    - Generation of random puzzles
- Difficulty levels: Easy, Medium, Hard, Expert, Extreme
    - Generation of puzzles according to difficulty level
- UX & GUI
    - Highlight
        - row, cell, box
        - similar numbers
    - Keyboard input
    - Erase cell functionality
    - Note-taking mode
        - Update notes on user input
    - Timer
        - Stop/restart timer
    - Undo functionality
        - Restore notes
    - Mistakes counter

## 2.0

This version is based in improving the existing game logic, adding new functionalities to the game and increasing the
quality of the UX.

- Sudoku puzzles 12x12 and 16x16
    - Generation of random puzzles
    - Improve generation of puzzles by introducing threads
    - Improve logged information
- UX & GUI
    - Hints functionality
    - Scoring
- TextUI

## 3.0

This version is based in providing a visual editor for the game that allows the user to create their own puzzles.
